When we use a random hash function h(x)h(x)h(x) to map keys into hashing numbers, we could design a fully random one which has the following property:

P(h(x)=i)=1n \mathbb{P}(h(x) = i) = \frac{1}{n} P(h(x)=i)=n1​

However, wit the above property, we need to store a table of xxx values and their hash values. This would take at least O(m)O(m)O(m) space and O(m)O(m)O(m) query time to look up, making our whole quest for O(1)O(1)O(1) query time pointless. Therefore we need to find another property that will avoid collision and O(1)O(1)O(1) query time.

2-Universial Hash Function (low collision probability). A random hash function from h:U→[n]h: U \to [n]h:U→[n] is two universal if :

P[h(x)=h(y)]≤1n(1) \mathbb{P}[h(x) = h(y)] \leq \frac{1}{n} \tag{1} P[h(x)=h(y)]≤n1​(1)

One of the functions that are often used is the following one:

h(x)=(ax+b modp) mod n h(x) = (a x + b \ \mathrm{mod} p) \ \mathrm{mod} \ n h(x)=(ax+b modp) mod n

Pairwise Independent Hash Function. A random hash function from h:U→[n]h: U \to [n]h:U→[n] is pairwise independence if for all i,j∈[n]i, j \in [n]i,j∈[n]:

P[h(x)=i∩h(y)=j]=1n2(2) \mathbb{P}[h(x) = i \cap h(y) = j] = \frac{1}{n^2} \tag{2} P[h(x)=i∩h(y)=j]=n21​(2)

We can show that pairwise hash function are 2-universal. Now, for i∈[n]i \in [n]i∈[n], it has n−1n-1n−1 pairs and one pair with itself, therefore:

P[h(x)=h(y)]=∑j=1nP[h(x)=i∩h(y)=j]=n⋅1n2=1n \begin{aligned} \mathbb{P}[h(x) = h(y)] & = \sum_{j=1}^n \mathbb{P}[h(x) = i \cap h(y) = j] \\ & = n \cdot \frac{1}{n^2} = \frac{1}{n} \end{aligned} P[h(x)=h(y)]​=j=1∑n​P[h(x)=i∩h(y)=j]=n⋅n21​=n1​​

Expected value
Informally, the expected value is the weighted arithmetic mean of a large number of independently selected outcomes of a random variable. This means we have: one random variable and a countable set of possible outcomes. For example, weight of human being follows the normal distribution: random variable is weight of human being, a countable set of possible outcomes is the set of all possible values of weight (some people are fit, some are slim, some are overweight). At last, we need to calculate the weighted arithmetic mean.

For the example of flipping coins, the random variable is HHH (the number of heads), the countable set of possible outcomes is the set of all possible values h∈[0,n]h \in [0, n]h∈[0,n]. We know each trial follows the binomial distribution, then we can write down the formula for the expected value.

Before we present Hoeffding’s inequality, we will learn a new concept called symmetric random variable.

A random variable ZZZ is symmetric if ZZZ and −Z-Z−Z have the same distribution. A simple example of a symmetric random variable is the well know symmetric Bernoulli, which takes values −1-1−1 and +1+1+1 with equal probability 1/21/21/2 each, i.e,

P[Z=1]=P[Z=−1]=1/2\mathbb{P}[Z = 1] = \mathbb{P}[Z = -1] = 1/2P[Z=1]=P[Z=−1]=1/2

For a usual Bernoulli random variable (XXX) with two possible values 1,01, 01,0 and parameter 1/21/21/2, we can transfer it into a symmetric Bernoulli random variable by setting

Z=2X−1Z = 2X - 1Z=2X−1

Hoeffding’s Inequality Let X1,⋯ ,XNX_1, \cdots, X_NX1​,⋯,XN​ be independent Bernoulli random variables, and let a=(a1,⋯ ,aN)∈RNa = (a_1, \cdots, a_N) \in \mathbb{R}^Na=(a1​,⋯,aN​)∈RN. Then, for any t≥0t \geq 0t≥0, we have

P[∑i=1NaiXi≥t]≤exp⁡(−t22∣∣a∣∣22)(17) \mathbb{P} \left [\sum_{i=1}^N a_i X_i \geq t \right ] \leq \exp \left ( - \frac{t^2}{2||a||_2^2} \right ) \tag{17} P[i=1∑N​ai​Xi​≥t]≤exp(−2∣∣a∣∣22​t2​)(17)

Remark: sometimes, you see the above formula for two-side inequality with absolute symbol. For the proof, you can find it here or in the book by Vershynin.

Let XiX_iXi​ be the random variable of mapping head or tail of flipping the coin into 1,01, 01,0, we are interested in the value of the number of heads in N=100N = 100N=100 trials. To use equation (17), we need to transform our random variable into symmetric one by setting

Yi=2(Xi−12)Y_i = 2(X_i - \frac{1}{2})Yi​=2(Xi​−21​)

Then we can have

P[∑i=1NaiXi>t]=P[∑i=1Nai(Yi2+12)>t]=P[∑i=1NYi>2t−N]ai=1 ∀i∈N≤exp⁡(−(2t−N)22N)∣∣a∣∣22=N \begin{aligned} \mathbb{P} \left [ \sum_{i=1}^N a_i X_i > t \right ] &= \mathbb{P} \left[ \sum_{i=1}^N a_i \left(\frac{Y_i}{2} + \frac{1}{2}\right) > t \right ] \\ &= \mathbb{P} \left[ \sum_{i=1}^N Y_i>2t-N \right ] \quad a_i = 1 \ \forall i \in N \\ & \leq \exp\left( - \frac{(2t-N)^2}{2 N}\right) \quad ||a||_2^2 = N \end{aligned} P[i=1∑N​ai​Xi​>t]​=P[i=1∑N​ai​(2Yi​​+21​)>t]=P[i=1∑N​Yi​>2t−N]ai​=1 ∀i∈N≤exp(−2N(2t−N)2​)∣∣a∣∣22​=N​
